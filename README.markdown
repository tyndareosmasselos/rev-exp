# Steps to run the project

1. Install node.js latest version
2. Install Angular -> npm install -g @angular/cli
2. Install Nest ->  npm i -g @nestjs/cli

# Run
## For Nest (API)

> cd/rev-exp-api
> npm i
> nest run

## For Angular (front-end)

> cd/rev-epi-web
> npm i
> ng serve

## Chrome or other web clients
> http://localhost:4200
